THYGRION presents a new intro - 

  - 3 D   O V E R L O A D -

Released February 2005!

It features {
	Milkshape 3D ASCII Models
	Perspective-Correct Texture Mapping
	Per-Pixel Shading
	Shadows
	Spherical Reflection Mapping
}

Credits {
	Code: Thygrion
	GFX: Thygrion, Clyde Radcliffe, Unknown
	Music: Unknown
}

The demo runs in 320x240x32. For fullscreen just run the program, for windowed run 'windowed.bat'.
Hit space to skip parts, or just wait 45 seconds. Hit 'F' for FPS info. ESCAPE quits.

Minimum requirements {
	400 mhz processor
	4 mb gfx card
	64 mb RAM
}

Enjoy!!!
